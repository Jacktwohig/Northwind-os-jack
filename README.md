# Northwind OS

Internal command center for Northwind Coffee — dashboard + inquiry triage workflow.

## Run locally

```bash
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173).

## What's inside

- **Dashboard** — revenue KPIs, weekly trend line, region breakdown, product mix, pipeline opportunity estimate, and accounts table
- **Triage** — 42 inbound wholesale inquiries scored Hot / Warm / Cold, with overdue flagging and persistent status actions (localStorage)

Built with React + Vite + TypeScript + Tailwind CSS + Recharts.
